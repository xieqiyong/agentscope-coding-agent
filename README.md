# Java AgentScope Coding Agent

This project is a Java + AgentScope based web coding agent. The product target is a browser-based coding assistant similar in spirit to opencode, but built around Java AgentScope, Spring Boot, SSE, tool calling, sandbox governance, user confirmation, and a first-class memory system.
